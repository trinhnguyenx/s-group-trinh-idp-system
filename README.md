

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## Running migrations
# Chạy migration:run
$ npm run migration:run

# Tạo migration mới
$ npm run migration:create -- tên_migration

# Tạo migration tự động từ mô hình hiện tại
$ npm run migration:generate -- tên_migration

# Revert migration
$ npm run migration:revert


## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
